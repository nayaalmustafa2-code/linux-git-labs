# Lab Report

## أهم الأوامر التي تم تنفيذها
- mkdir
- cd
- git init (لاحقًا)
- git add / commit (لاحقًا)

## الفرق بين > و >>
> يكتب ويستبدل المحتوى  
>> يضيف على نفس الملف بدون حذف القديم

## متى نستخدم source بدل ./script.sh؟
نستخدم source عندما نريد تنفيذ السكربت داخل نفس الشِل

## الفرق بين && و ||
&& يعني نفّذ الأمر الثاني إذا الأول نجح  
|| يعني نفّذ الأمر الثاني إذا الأول فشل

## لماذا نستخدم الفروع في Git؟
لتجربة تغييرات بدون التأثير على النسخة الأساسية

## ما وظيفة ssh-agent؟
إدارة مفاتيح SSH لتسجيل الدخول بدون كلمة مرور
